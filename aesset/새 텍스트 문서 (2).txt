# 특별실 예약 웹사이트 제작 프롬프트

학교 내 특별실 사용의 비효율성을 해결하기 위한 웹 기반 특별실 예약 시스템을 제작한다.

## 프로젝트명

학교 특별실 예약 및 관리 시스템

## 프로젝트 목적

현재 동아리 활동 시간이나 이동수업 시간에 특별실 사용 여부를 확인하기 위해 직접 특별실을 방문하거나 담당 교사에게 문의해야 하는 문제가 있다. 또한 여러 학급이나 동아리가 같은 특별실을 동시에 사용하려고 하여 일정 충돌이 발생하기도 한다.

이를 해결하기 위해 학생과 교사가 웹사이트에서 특별실의 사용 현황을 실시간으로 확인하고 예약할 수 있는 시스템을 구축한다.

---

## 사용자 유형

### 학생

* 특별실 사용 가능 여부 조회
* 특별실 예약 신청
* 자신의 예약 내역 확인
* 예약 취소 및 수정

### 교사

* 특별실 예약
* 학생 예약 승인 또는 거절
* 예약 현황 조회
* 특별실 이용 통계 확인

### 관리자

* 특별실 추가, 수정, 삭제
* 사용자 관리
* 예약 승인 및 강제 취소
* 시스템 전체 관리

---

## 주요 기능

### 1. 로그인 기능

* 학교 계정으로 로그인
* 학생, 교사, 관리자 권한 구분
* 역할에 따라 접근 가능한 메뉴 자동 변경

### 2. 대시보드

* 오늘 사용 중인 특별실 목록 표시
* 현재 비어 있는 특별실 표시
* 오늘의 예약 현황 요약 제공
* 사용률 통계 표시

### 3. 특별실 목록 페이지

다음 정보를 카드 형식으로 표시한다.

* 특별실 이름
* 위치(예: 본관 3층)
* 수용 인원
* 현재 사용 상태
* 다음 예약 시간
* 예약 버튼

예시 특별실:

* 컴퓨터실
* 과학실
* 음악실
* 미술실
* 영어전용실
* 메이커실
* 방송실

### 4. 예약 시스템

예약 시 다음 정보를 입력한다.

* 사용자 이름
* 학년 및 반
* 사용 목적
* 특별실 선택
* 사용 날짜
* 시작 시간
* 종료 시간
* 참가 인원

예약 시간 중복 시 자동으로 예약을 차단한다.

### 5. 달력 기능

* 월간 캘린더 제공
* 날짜별 예약 현황 표시
* 시간표 형식의 예약 현황 제공
* 사용 중인 특별실은 빨간색
* 예약 가능한 특별실은 초록색

### 6. 승인 시스템

* 학생 예약은 담당 교사 승인 후 확정
* 교사는 승인 또는 거절 가능
* 승인 결과는 알림으로 전달

### 7. 알림 기능

* 예약 승인 알림
* 예약 거절 알림
* 사용 시작 10분 전 알림
* 예약 종료 10분 전 알림

### 8. 관리자 페이지

* 특별실 추가 및 수정
* 예약 내역 조회
* 월별 사용 통계 확인
* 가장 많이 사용된 특별실 순위 제공

---

## UI 및 디자인 요구사항

* 학교 행정 시스템과 유사한 깔끔한 디자인
* 반응형 웹 디자인 적용
* 모바일, 태블릿, PC 지원
* 직관적인 인터페이스 제공
* 예약 현황을 색상으로 쉽게 구분 가능하도록 구현

추천 색상:

* 파란색: 예약 가능
* 빨간색: 사용 중
* 노란색: 승인 대기
* 회색: 사용 불가

## 최종 목표

학생과 교사가 특별실 사용 여부를 실시간으로 확인하고 예약할 수 있도록 하여 불필요한 이동과 문의 과정을 줄이고, 학교 시설을 효율적으로 운영할 수 있는 웹 기반 특별실 예약 플랫폼을 구축한다.
